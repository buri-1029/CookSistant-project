# CookSistant

> _자취생을 위한 평가기반 레시피 추천 서비스_

<br/>

---

- **개발 기간** : 2021.03.01 ~ 2021.04.09 | **인원** : 4명 
- **주요 내용** : 자취생을 위하여 사용자가 레시피에 대하여 **최근 평가한 내용**과 **본인과 비슷한 평가를 한 사용자들의 평가**를 기반으로 레시피를 추천해주는 서비스
